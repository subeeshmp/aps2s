Welcome to aps2s's documentation!
=================================
**aps2s** is a python cli utility (and library) for generating and manipulating input and configuration files to run *SKRIPS* model developed at *KAUST*. *SKRIPS* is coupled model with *WRF* as its atmosphere and *MITgcm* as its ocean component.

.. note::
    This project is under active development.
